Alternar entre Aplicativos

  1. **Pressione o botão de aplicativos recentes:** Este botão geralmente está localizado na parte inferior da tela e exibe uma lista dos aplicativos abertos.
  2. **Escolha um aplicativo:** Toque no aplicativo desejado na lista.
  3. **Fechar aplicativos:** Para fechar, deslize o aplicativo para cima ou toque no "X" (dependendo do dispositivo).
  4. **Dica:** Alternar rapidamente entre aplicativos é útil para multitarefas.
